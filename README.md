# AI Product Builder

My journey to becoming a technical product builder for AI systems.

## Week 1 — Python Fundamentals
- Variables, functions, loops, error handling
- Lists, dicts, list-of-dicts
- File I/O + JSON (persistent storage)
- HTTP requests with `requests`
